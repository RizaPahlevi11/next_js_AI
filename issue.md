# Feature: Implementasi Login Screen dari Stitch AI

## 1. Deskripsi
Tugas ini ditujukan untuk mengimplementasikan antarmuka (UI) **Login Screen** berdasarkan referensi desain yang sudah saya buat secara terpisah menggunakan **Stitch AI**.

## 2. Tujuan
- Menerapkan _layout_ dan komponen UI (form email, password, tombol login/register) persis seperti desain pada Stitch AI.
- Menyesuaikan _style_ (warna, tipografi, _spacing_, dan _corner radius_) agar seirama dengan *design system* yang didefinisikan di proyek Stitch bersangkutan.
- Menggabungkan UI yang sudah diimplementasikan/di-_generate_ dengan logika autentikasi (NextAuth) yang telah kita bangun sebelumnya.

## 3. Langkah Kerja
1. Periksa proyek di **StitchMCP**. Ambil *Screen ID* atau detail dari proyek yang ada.
2. Generate kode komponen UI berdasarkan tangkapan layar / teks *prompt* desain terkait (misalnya dengan mengeksekusi *tool* integrasi Stitch).
3. Sesuaikan dengan standar `shadcn/ui` dan Tailwind CSS proyek jika diperlukan penyesuaian teknis secara manual.
4. Terapkan pada rute `/login`.
